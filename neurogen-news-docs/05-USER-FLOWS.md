# Пользовательские сценарии Neurogen.News

## Обзор функционала авторизованного пользователя

На основе детального анализа VC.ru с авторизованным пользователем выявлены следующие функции и сценарии.

---

## 1. Меню пользователя

### Структура dropdown-меню

При клике на аватар в header открывается меню:

```
┌─────────────────────────────────────┐
│  Мой профиль                        │
│  ┌───────────────────────────────┐ │
│  │ [Avatar] ИМЯ ПОЛЬЗОВАТЕЛЯ 💎  │ │
│  │          @username            │ │
│  └───────────────────────────────┘ │
│                                     │
│  ✏️ Черновики                       │
│  🔖 Закладки                        │
│  🏆 Ачивки                          │
│  💰 Донаты          [Подключить]    │
│  ⚙️ Настройки              [🌓]    │
│                                     │
│  💎 Подписка Plus                   │
│                                     │
│  ↪️ Выйти                           │
└─────────────────────────────────────┘
```

### Элементы меню

| Элемент | URL | Описание |
|---------|-----|----------|
| Мой профиль | /username | Ссылка на профиль |
| Черновики | /drafts | Неопубликованные статьи |
| Закладки | /bookmarks | Сохранённые материалы |
| Ачивки | /achievements/username | Достижения пользователя |
| Донаты | /donations/registration | Подключение приёма донатов |
| Настройки | /settings | Настройки аккаунта |
| Подписка Plus | /plus | Премиум подписка |

## 3. Профиль пользователя (/username)

### Структура профиля

```
┌─────────────────────────────────────────────────────────────────┐
│  [+ Добавить обложку]                                          │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                     ОБЛОЖКА ПРОФИЛЯ                     │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  [Avatar 96px]                              [📊] [⚙️]           │
│                                      (статистика) (настройки)   │
│  ИМЯ ПОЛЬЗОВАТЕЛЯ 💎                                           │
│  @username                                                     │
│                                                                 │
│  с DD.MM.YYYY                                                  │
│                                                                 │
│  [N подписчиков] [N подписок]                                  │
│                                                                 │
│  ─────────────────────────────────────────────────────────────│
│  [Посты]  [Комментарии]                                        │
│  ─────────────────────────────────────────────────────────────│
│                                                                 │
│  [Свежее ▼]                                                    │
│                                                                 │
│  [Список постов/комментариев]                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Функции для владельца профиля

- **Добавить обложку** — загрузка изображения обложки
- **Редактировать аватар** — смена аватара
- **📊 Статистика** — аналитика по постам и подписчикам
- **⚙️ Настройки** — быстрый переход в настройки

### Вкладки профиля

| Вкладка | URL | Содержимое |
|---------|-----|------------|
| Посты | /username | Все опубликованные посты |
| Комментарии | /username/comments | Все комментарии пользователя |

## Рекомендации для Neurogen.News

### Ключевые особенности для внедрения

1. **Система достижений (Ачивки)**
   - Геймификация активности пользователей
   - Публичные бейджи в профиле
   - Прогресс к следующему достижению

2. **Умные уведомления**
   - Гранулярные настройки по типам
   - Email + Push + In-app
   - Группировка похожих уведомлений

3. **Персонализированная лента**
   - "Моя лента" из подписок
   - Рекомендации на основе интересов
   - Фильтрация нежелательного контента

4. **Монетизация авторов**
   - Система донатов
   - Premium-контент (для Plus подписчиков)
   - Партнёрская программа

5. **Профиль как визитная карточка**
   - Кастомизация (обложка, описание)
   - Статистика и аналитика
   - Ачивки и бейджи
