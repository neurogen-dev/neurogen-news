# Функциональные требования Neurogen.News

## Навигация и структура

### Главное меню (левый сайдбар)

По аналогии с DTF/VC — фиксированный сайдбар слева:

| Секция | Элементы | Описание |
|--------|----------|----------|
| **Основные** | Популярное | Алгоритмическая лента (вовлечённость + время) |
| | Свежее | Хронологическая лента новых публикаций |
| | Моя лента | Подписки пользователя (авторы + темы) |
| | 🟢 **С чего начать** | Гайды для новичков |
| | Сообщения | Личные сообщения и уведомления |
| | Рейтинг | Топ авторов и компаний |
| **По применению** | 💬 Чат-боты | ChatGPT, Claude, Gemini |
| | 🎨 Изображения | Midjourney, Stable Diffusion, DALL-E |
| | 🎬 Видео | Sora, Runway, генерация видео |
| | 🎵 Музыка | Suno, Udio, озвучка |
| | ✍️ Тексты | Копирайтинг, перевод |
| | 💻 Код | Cursor, Copilot, разработка |
| **По формату** | 📖 Гайды | Пошаговые инструкции |
| | 🔍 Обзоры | Сравнения сервисов |
| | 💡 Промпты | Библиотека промптов |
| | 📰 Новости | Анонсы и тренды |
| | 🏢 Бизнес | Кейсы внедрения |
| | ❓ Вопросы | Q&A сообщества |
| | 🛠 Инструменты | Каталог сервисов |
| **О проекте** | О проекте | Информационная страница |
| | Правила | Правила сообщества |
| | Реклама | Рекламные возможности |
| | Приложения | Ссылки на мобильные приложения |

### Шапка (header)

```
┌────────────────────────────────────────────────────────────────────┐
│  [Logo]                        [🔍]  [✏️ Написать]  [Avatar/Войти] │
└────────────────────────────────────────────────────────────────────┘
```

**Элементы:**
- Логотип — клик возвращает на главную
- Поиск — открывает поисковый оверлей
- Кнопка "Написать" — переход в редактор (или quick-editor)
- Аватар пользователя / кнопка "Войти"

### Меню пользователя (по аналогии с VC.ru)

При клике на аватар открывается dropdown-меню:

```
┌────────────────────────────────┐
│  [Avatar]                      │
│  Имя Пользователя              │

│  👤 Мой профиль               │

├────────────────────────────────┤
```

**Разделы меню:**
| Раздел | Описание |
|--------|----------|
| Мой профиль | Переход на публичную страницу профиля |

```
┌────────────────────────────────────────────────────────────────────┐
│  [Avatar] Автор Name • 💬 Чат-боты • 2ч [✏️]  [Подписаться]       │

│                                                                    │
│  Лид текст/превью статьи с ограничением символов...               │
│                                                                    │
│  ┌──────────────────────────────────────┐                          │
│  │                                      │                          │
│  │         Обложка статьи               │                          │
│  │                                      │                          │
│  └──────────────────────────────────────┘                          │

├────────────────────────────────────────────────────────────────────┤
│  [💬 128]  [🔖 15]  [↗️]  [···]  [👁 12.5K]                         │
├────────────────────────────────────────────────────────────────────┤
│  "Лучший комментарий из обсуждения..."                            │
│  [2 ответа на пост]                                                │
└────────────────────────────────────────────────────────────────────┘
```

### Компоненты карточки

1. **Шапка карточки**
   - Аватар автора
   - Имя автора (кликабельно)
   - Категория/подсайт (кликабельно)
   - Время публикации
   - Индикатор редактирования
   - Кнопка "Подписаться"

2. **Метаинформация (NEW)**
   - **Бейдж уровня:** 🟢 Для новичков / 🟡 Продвинутое / 🔴 Для бизнеса
   - **Тип контента:** 📖 Гайд / 🔍 Обзор / 💡 Промпты / 📰 Новость
   - **Время чтения:** "5 мин" — помогает выбрать контент

3. **Контент**
   - Заголовок (H2)
   - Бейдж "Материал редакции" (для редакционного контента)
   - Лид-текст (ограничение ~300 символов)
   - Обложка (соотношение 16:9)
   - Кнопка "Показать полностью"

4. **Панель реакций**
   - Эмодзи-реакции с счётчиками
   - Кнопка добавления реакции

5. **Панель действий**
   - Комментарии (счётчик)
   - Закладки (счётчик) — добавляет в закладки пользователя
   - Поделиться
   - Меню (···)
   - Просмотры

6. **Превью комментария**
   - Лучший комментарий
   - Количество ответов

## Быстрый редактор (Quick Editor)

По аналогии с VC.ru — возможность быстро создать пост прямо из ленты:

```
┌────────────────────────────────────────────────────────────────────┐
│  [Avatar]                                                          │
│  ┌──────────────────────────────────────────────────────────────┐ │
│  │ О чём хотите рассказать?                                     │ │
│  │                                                              │ │
│  │ (textarea)                                                   │ │
│  │                                                              │ │
│  └──────────────────────────────────────────────────────────────┘ │
│                                                                    │
│  [📎 Файл] [🖼 Изображение] [🔗 Ссылка]  [Развернуть ↗]           │
│                                                                    │
│  Категория: [Выбрать ▼]              [Опубликовать]               │
└────────────────────────────────────────────────────────────────────┘
```

**Функционал:**
- Textarea для быстрого ввода текста
- Прикрепление файлов и изображений
- Вставка ссылок
- Кнопка "Развернуть" → переход в полноэкранный редактор
- Выбор категории
- Быстрая публикация

### Блоки контента (Rich Text)

#### Стандартные блоки

| Тип блока | Описание |
|-----------|----------|
| Текст | Параграфы, заголовки (H2-H4), списки |
| Изображение | Одиночное изображение с подписью |
| Галерея | Слайдер из нескольких изображений |
| Видео | YouTube, Vimeo, VK Video, Rutube |
| Embed | Twitter, Telegram, Instagram посты |
| Цитата | Выделенный блок цитаты |
| Код | Подсветка синтаксиса (highlight.js) |
| Спойлер | Скрытый контент с раскрытием |
| Разделитель | Горизонтальная линия |
| Таблица | Табличные данные |

#### Практические блоки (NEW)

| Тип блока | Описание | Пример |
|-----------|----------|--------|
| **Промпт** | Копируемый промпт с кнопкой "Скопировать" | Промпт для ChatGPT с одним кликом |

## Система комментариев

### Функции

1. **Создание комментария**
   - Текстовый редактор с форматированием
   - Вложение изображений
   - GIF через GIPHY интеграцию
   - Упоминания пользователей (@username)
   - Донаты автору (₽)

2. **Вложенность**
   - Древовидная структура
   - Ответы на комментарии
   - Сворачивание веток
   - "Показать N комментариев"

3. **Сортировка**
   - Лучшие (по рейтингу)
   - Свежие (хронологически)
   - Старые (хронологически)
   - Популярные (по активности)

4. **Модерация**
   - Редактирование (с пометкой)
   - Удаление
   - Жалоба
   - Скрытие от автора

5. **Подписка на обсуждение**
   - Уведомления о новых комментариях

### Панель инструментов (по аналогии с VC.ru)

| Кнопка | Функция |
|--------|---------|
| **B** | Жирный текст (Bold) |
| **I** | Курсив (Italic) |
| **U** | Подчёркнутый (Underline) |
| **S** | Зачёркнутый (Strikethrough) |
| 🔗 | Вставка ссылки |
| 🖼 | Вставка изображения |
| 📎 | Вложение файла |
| **H2** | Заголовок 2 уровня |
| **H3** | Заголовок 3 уровня |
| " | Цитата (Blockquote) |
| </> | Блок кода |
| • | Маркированный список |
| ▼ | Спойлер |
| ··· | Дополнительные опции |

### Функции редактора

1. **Форматирование текста**
   - Bold, Italic, Underline, Strikethrough
   - Заголовки H2, H3
   - Списки (маркированные, нумерованные)
   - Цитаты
   - Код (inline и блок)
   - Ссылки
